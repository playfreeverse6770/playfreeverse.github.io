PlayFreeverse

PlayFreeverse is an indie game development project focused on creating creative and accessible game experiences using Unity.

Vision

Our long-term goal is to build original games, interactive experiences, and reusable Unity assets. We are currently in early development stages, focusing on experimentation, learning, and prototyping.

Projects

PlayFreeverse is currently working on early-stage game development and AI-assisted Unity tools. All projects are in active research or prototyping phases.

Blue Ocean Scene AI (In Development)

Blue Ocean Scene AI is an experimental AI-assisted Unity system designed to analyze, generate, and improve game scenes and systems.

  • Scene analysis and structured data representation
  • AI-assisted generation of missing systems and logic
  • Validation, fixing, and iterative workflows
  • AI-first architecture designed for extensibility

Game Concepts (Early Development)

  • Freeverse — An open-world game concept focused on exploration and systemic gameplay.
  • Fly Soccer — A two-player football game with fast-paced and experimental mechanics.
  • PolyRealm Legends — A strategic game concept set in a stylized fantasy world.
