PlayFreeverse

Independent Game Development & Editor Tools

PlayFreeverse is an independent development project focused on building original games and practical editor tools through hands-on production experience.

Our Story

PlayFreeverse started as a game development initiative. While working on an open-world game prototype, we encountered repeated limitations in scene inspection, debugging, and iteration workflows.

Instead of relying solely on existing tools, we began creating internal editor extensions to better understand, analyze, and safely modify complex scenes. Over time, these internal tools became stable, reusable, and valuable beyond a single project.

This process led to the creation of BOSA — a growing set of editor tools designed from real production needs.

Current Focus: BOSA – Base

BOSA – Base is the first public release of our internal tooling. It provides essential editor-side utilities for scene analysis, structured JSON inspection, object placement, and safe scene backups.

This version is intentionally lightweight and focused. It is designed for developers who want visibility and control without introducing risk to their projects.

BOSA – Base is an editor-only toolset and does not include runtime components.

Game Projects

Freeverse (In Development)

Freeverse is an open-world game concept focused on exploration, systemic gameplay, and large-scale scene composition. The complexity of this project directly influenced the design of our internal editor tools.

Experimental Concepts

Alongside Freeverse, we experiment with smaller gameplay ideas and prototypes, such as fast-paced sports mechanics and stylized strategy concepts. These experiments help validate tools across different genres and scales.

Tooling Philosophy

Our tools are built with the same mindset as our games: clarity over complexity, safety over shortcuts, and extensibility over rigid design.

Every public tool we release originates from real usage inside a working project. We avoid speculative features and focus on workflows that have already proven useful.
