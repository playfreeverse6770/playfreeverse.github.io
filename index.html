PlayFreeverse

PlayFreeverse is an indie game development project focused on creating creative and accessible game experiences using Unity.

Vision

Our long-term goal is to build original games, interactive experiences, and reusable Unity assets. We are currently in early development stages, focusing on experimentation, learning, and prototyping.

Roadmap

  • Game prototypes and MVP builds
  • Unity tools and reusable assets
  • Public demos and WebGL releases
